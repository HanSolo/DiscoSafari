# DiscoSafari

A little Safari extension that helps you discovering and download a JDK of your choice.
It makes use of the foojay Disco API and supports the following distributions:

- AdoptOpenJDK
- Corretto
- Dragonwell
- Liberica
- Microsoft Build of OpenJDK
- OJDK Build
- OpenLogic
- Oracle OpenJDK
- SAP Machine
- Temurin
- Trava
- Zulu
